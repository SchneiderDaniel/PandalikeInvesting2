## Setup your env variables
You need to set the following env. variables.

    RECAPTCHA_PUBLIC_KEY
    RECAPTCHA_PRIVATE_KEY
    SECRET_KEY
    DATABASE_URI (example: sqlite:///site.db)
    ADMIN_USER
    ADMIN_EMAIL
    ADMIN_PASSWORD
    CONFIG_MODE (Debug/Production)
 
## Run the application
    .\env\Scripts\activate
    python run.py
